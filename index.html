Binário para Decimal

Como funciona?

  • Separe cada dígito do numero binario

  • Utilize a Formula: n x bi em cada dígito iniciando da direta para a esquerda

  • n = digito, b = base do binario(base 2), i = a posição do dígito

Vamos pegar o seguinte valor como exemplo:

1 0 1 1 1

Aplicando a fórmula em cada dígito da direita para a esquerda temos:

(1x24) + (0x23) + (1x22) + (1x21) + (1x20)

16 + 0 + 4 + 2 + 1

Com os cálculos obtemos o valor: 23
